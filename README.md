# CodingTools
用PyQt5写的一个关于编码的小工具

## 支持的功能
简单实现了如下功能的编码与解码，可以选择UTF-8或GBK字符集
- URL  
- base64  
- hex  
- Unicode  
- 时间戳转换  

图片文件形式的二维码识别  
文本、文件md5、sha256计算  
简单文本替换
